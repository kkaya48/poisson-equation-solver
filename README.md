# Numerical Poisson Equation Solver in MATLAB

This project implements a numerical solver for a two-dimensional Poisson equation using MATLAB.

The solver discretizes the computational domain, identifies boundary and interior points, assembles the corresponding linear system, and visualizes the numerical solution as a surface plot.

## Main Features
- 2D Poisson equation solver
- Finite-difference discretization
- Boundary and interior point handling
- Linear-system assembly
- Numerical solution visualization
- Surface plot of the computed result

## Technologies
- MATLAB
- Numerical methods
- Finite differences
- Linear algebra
- Scientific visualization

## Purpose

The project was developed as part of my work in numerical mathematics and scientific computing. It focuses on translating a mathematical boundary value problem into a structured computational solution.
